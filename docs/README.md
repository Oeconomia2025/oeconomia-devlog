# Oeconomia Updates

Weekly development reports from the Oeconomia DeFi ecosystem.

Oeconomia is a self-funded, multi-protocol DeFi ecosystem built from scratch, including:

- **OEC Token** -- 500M supply with presale and staking emissions
- **Eloqura DEX** -- AMM DEX with TradingView Advanced Charts
- **Alluria** -- Liquity-style CDP lending with ALUD stablecoin
- **Staking dApp** -- Live on testnet at staking.oeconomia.io
- **DAO Hub** -- Governance dashboard with presale and roadmap
- **OECsplorer** -- Block explorer for Oeconomia protocols

Each week, a detailed devlog and a short public update are published here.
